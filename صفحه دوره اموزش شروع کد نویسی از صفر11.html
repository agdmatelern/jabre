مقدمه‌ای بر تفکر الگوریتمی

تفکر برنامه‌نویسی (Computational Thinking) یک مهارت اساسی در دنیای دیجیتال امروز است که به شما توانایی حل مسائل پیچیده را با رویکردی سیستماتیک می‌دهد. این ماژول پایه و اساس تمام مهارت‌های برنامه‌نویسی شما را تشکیل می‌دهد.

در این ماژول یاد می‌گیرید که چگونه مسائل پیچیده را به بخش‌های کوچکتر و قابل مدیریت تقسیم کنید، الگوهای موجود در مسائل را شناسایی کنید، و راه‌حل‌های کارآمدی طراحی نمایید که کامپیوترها بتوانند آنها را اجرا کنند. این مهارت نه تنها در برنامه‌نویسی، بلکه در حل مسائل روزمره زندگی نیز به کار می‌آید.

مفاهیم کلیدی تفکر برنامه‌نویسی

  • تجزیه (Decomposition): تقسیم مسئله بزرگ به زیرمسائل کوچکتر و قابل مدیریت
  • تشخیص الگو (Pattern Recognition): شناسایی شباهت‌ها و تفاوت‌ها در مسائل برای استفاده از راه‌حل‌های موجود
  • انتزاع (Abstraction): تمرکز بر اطلاعات و ویژگی‌های ضروری و نادیده گرفتن جزئیات غیرضروری
  • الگوریتم‌سازی (Algorithm Design): طراحی گام‌به‌گام دستورالعمل‌ها برای حل مسئله
  • بهینه‌سازی (Optimization): بهبود راه‌حل‌ها برای کارایی بهتر و مصرف منابع کمتر

نکته مهم: تفکر برنامه‌نویسی یک مهارت ذهنی است که با تمرین و تکرار توسعه می‌یابد. سعی کنید مسائل روزمره خود را با این رویکرد تحلیل کنید. مثلاً هنگام پختن غذا، مراحل کار را به صورت الگوریتمی در ذهن خود طراحی کنید.

طراحی سیستم ورود کاربر

  • پس از ۳ بار تلاش ناموفق، حساب کاربری به مدت ۱۰ دقیقه قفل شود

آشنایی با محیط‌های توسعه

یک محیط توسعه یکپارچه (IDE) ابزاری ضروری برای برنامه‌نویسان حرفه‌ای است که تمام امکانات لازم برای توسعه نرم‌افزار را در یک محیط واحد فراهم می‌کند. در این ماژول با بهترین ابزارهای توسعه آشنا می‌شوید و یاد می‌گیرید چگونه محیط کاری خود را بهینه کنید.

مهم‌ترین ابزارهای توسعه

  • VS Code: ویرایشگر کد قدرتمند و سبک با هزاران افزونه کاربردی
  • PyCharm: محیط توسعه تخصصی برای برنامه‌نویسی پایتون
  • IntelliJ IDEA: IDE حرفه‌ای برای توسعه جاوا و سایر زبان‌های JVM
  • Git و GitHub: سیستم کنترل نسخه برای مدیریت کد و همکاری تیمی
  • Docker: پلتفرم کانتینری‌سازی برای ایجاد محیط‌های توسعه یکسان

نکته حرفه‌ای: برای افزایش بهره‌وری، افزونه‌های زیر را در VS Code نصب کنید: ESLint (برای بررسی کد جاوااسکریپت)، Prettier (برای فرمت‌بندی خودکار کد)، Python (برای پشتیبانی از پایتون)، GitLens (برای مشاهده تاریخچه تغییرات)، Live Server (برای اجرای سریع کدهای وب).

راه‌اندازی محیط توسعه پایتون

پروژه‌ای را با مشخصات زیر ایجاد و پیکربندی کنید:

  • ایجاد محیط مجازی پایتون با نام venv
  • نصب پکیج‌های ضروری: numpy, pandas, matplotlib
  • پیکربندی VS Code برای استفاده از محیط مجازی ایجاد شده
  • ایجاد یک فایل requirements.txt برای مدیریت وابستگی‌ها
  • تنظیم لینتر و فرمتر برای استانداردسازی کد
# ایجاد محیط مجازی

pip freeze > requirements.txt

انواع داده و متغیرها

در برنامه‌نویسی، داده‌ها انواع مختلفی دارند که هر کدام برای هدف خاصی استفاده می‌شوند. آشنایی با انواع داده اولین قدم در یادگیری برنامه‌نویسی است.

انواع داده اصلی

  • اعداد صحیح (Integer): برای ذخیره اعداد بدون اعشار
  • اعداد اعشاری (Float): برای ذخیره اعداد با دقت اعشار
  • رشته‌ها (String): برای ذخیره متن و کاراکترها
  • مقادیر بولی (Boolean): برای ذخیره مقادیر درست یا نادرست
  • لیست‌ها (List): برای ذخیره مجموعه‌ای از مقادیر مرتب
  • دیکشنری‌ها (Dictionary): برای ذخیره داده‌های کلید-مقدار

نکته مهم: انتخاب نوع داده مناسب برای متغیرها تأثیر مستقیمی بر عملکرد و خوانایی کد دارد. همیشه از مناسب‌ترین نوع داده برای ذخیره اطلاعات استفاده کنید.

عملگرهای اصلی در برنامه‌نویسی

عملگرها ابزارهایی برای انجام عملیات روی داده‌ها هستند. در زیر مهم‌ترین عملگرها در پایتون را مشاهده می‌کنید:

عملگرها در پایتون
# عملگرهای محاسباتی
جمع = 10 + 5       # 15
تفریق = 10 - 5     # 5
ضرب = 10 * 5       # 50
تقسیم = 10 / 5     # 2.0
تقسیم صحیح = 10 // 3  # 3
باقیمانده = 10 % 3   # 1
توان = 2 ** 3      # 8

# عملگرهای مقایسه‌ای
مساوی = 10 == 10   # True
نابرابر = 10 != 5  # True
بزرگتر = 10 > 5    # True
کوچکتر = 10 < 5    # False
بزرگتر مساوی = 10 >= 10  # True
کوچکتر مساوی = 5 <= 10   # True

# عملگرهای منطقی
و = True and False  # False
یا = True or False  # True
نقیض = not True     # False

تمرین: ماشین حساب ساده

پیاده‌سازی ماشین حساب

برنامه‌ای بنویسید که دو عدد و یک عملگر (+, -, *, /) از کاربر دریافت کند و نتیجه عملیات را نمایش دهد. برنامه باید موارد زیر را بررسی کند:

  • اعتبارسنجی ورودی‌ها (اعداد باید معتبر باشند)
  • در صورت تقسیم بر صفر، پیغام خطای مناسب نمایش داده شود
  • پس از نمایش نتیجه، از کاربر بپرسد که آیا می‌خواهد عملیات دیگری انجام دهد یا خیر
while True:
    try:
        num1 = float(input("عدد اول را وارد کنید: "))
        operator = input("عملگر (+, -, *, /) را وارد کنید: ")
        num2 = float(input("عدد دوم را وارد کنید: "))
        
        if operator == '+':
            result = num1 + num2
        elif operator == '-':
            result = num1 - num2
        elif operator == '*':
            result = num1 * num2
        elif operator == '/':
            if num2 == 0:
                print("خطا: تقسیم بر صفر امکان‌پذیر نیست!")
                continue
            result = num1 / num2
        else:
            print("عملگر نامعتبر!")
            continue
            
        print(f"نتیجه: {num1} {operator} {num2} = {result}")
        
    except ValueError:
        print("خطا: لطفاً عدد معتبر وارد کنید!")
    
    repeat = input("آیا می‌خواهید ادامه دهید؟ (بله/خیر): ")
    if repeat.lower() != 'بله':
        break

دستورات شرطی

دستورات شرطی به برنامه اجازه می‌دهند بر اساس شرایط مختلف، مسیرهای اجرایی متفاوتی را طی کند. این ساختارها پایه و اساس منطق برنامه‌نویسی هستند.

انواع دستورات شرطی

  • if: اجرای بلوک کد در صورت برقراری شرط
  • else: اجرای بلوک کد در صورت عدم برقراری شرط if
  • elif: بررسی شرط‌های متوالی در صورت عدم برقراری شرط‌های قبلی
  • شرط‌های تو در تو: استفاده از شرط‌های چند سطحی برای پیاده‌سازی منطق پیچیده

نکته حرفه‌ای: از شرط‌های پیچیده و تو در تو پرهیز کنید. اگر تعداد شرط‌های شما بیشتر از ۳ سطح شد، بهتر است از ساختار switch-case (در زبان‌های پشتیبانی کننده) یا دیکشنری‌های تابعی استفاده کنید.

پیاده‌سازی سیستم نمره‌دهی

برنامه‌ای بنویسید که نمره دانشجو را دریافت کرده و وضعیت تحصیلی او را تعیین کند:

سیستم نمره‌دهی
نمره = float(input("نمره دانشجو را وارد کنید: "))

if نمره >= 90:
    وضعیت = "A"
elif نمره >= 80:
    وضعیت = "B"
elif نمره >= 70:
    وضعیت = "C"
elif نمره >= 60:
    وضعیت = "D"
else:
    وضعیت = "F"

print(f"وضعیت تحصیلی: {وضعیت}")

حلقه‌های تکرار

حلقه‌ها برای تکرار یک بلوک کد به تعداد مشخص یا تا زمانی که شرط خاصی برقرار باشد استفاده می‌شوند.

انواع حلقه‌ها

  • for: برای تکرار روی عناصر یک دنباله (لیست، رشته، دیکشنری و...)
  • while: برای تکرار تا زمانی که شرط خاصی برقرار باشد
  • break: خروج زودرس از حلقه
  • continue: رد شدن به تکرار بعدی حلقه
مثال‌هایی از حلقه‌ها
# حلقه for
اعداد = [1, 2, 3, 4, 5]
مجموع = 0
for عدد in اعداد:
    مجموع += عدد
print(f"مجموع اعداد: {مجموع}")

# حلقه while
عدد = 1
while عدد <= 5:
    print(عدد)
    عدد += 1

# استفاده از break و continue
for عدد in range(1, 11):
    if عدد % 2 == 0:
        continue  # اعداد زوج را رد می‌کند
    if عدد > 7:
        break     # حلقه را متوقف می‌کند
    print(عدد)

ماژول 5: حلقه‌ها و تکرار

کار با حلقه‌های for و while برای تکرار عملیات

تمرینات: ۸ تمرین عملی
پروژه‌ها: ۴ پروژه پایانی

حلقه for در پایتون

حلقه for در پایتون برای تکرار بر روی عناصر یک دنباله (لیست، تاپل، رشته، دیکشنری و...) استفاده می‌شود. این حلقه بسیار قدرتمند و انعطاف‌پذیر است.

مثال‌های کاربردی حلقه for

کاربردهای حلقه for
# تکرار روی لیست
میوه‌ها = ["سیب", "موز", "گلابی"]
for میوه in میوه‌ها:
    print(میوه)

# تکرار روی رشته
برای حرف in "پایتون":
    print(حرف)

# تکرار با اندیس
برای اندیس, میوه in enumerate(میوه‌ها):
    print(f"اندیس {اندیس}: {میوه}")

# تکرار روی دیکشنری
کاربر = {"نام": "علی", "سن": 25, "شهر": "تهران"}
برای کلید, مقدار in کاربر.items():
    print(f"{کلید}: {مقدار}")

# حلقه تو در تو
برای i in range(3):
    برای j in range(2):
        print(f"({i}, {j})")

نکته حرفه‌ای: از لیست کامپری‌شن‌ها برای ایجاد لیست‌های جدید از روی لیست‌های موجود استفاده کنید. این روش هم کوتاه‌تر است و هم کارایی بهتری دارد.

حلقه while

حلقه while تا زمانی که شرط آن درست باشد، بلوک کد خود را اجرا می‌کند. این حلقه برای موقعیت‌هایی که تعداد تکرارها از قبل مشخص نیست مفید است.

مثال‌هایی از حلقه while
# مثال ساده
عدد = 1
while عدد <= 5:
    print(عدد)
    عدد += 1

# ورودی کاربر
مجموع = 0
while True:
    ورودی = input("عدد وارد کنید (خروج برای خاتمه): ")
    if ورودی == "خروج":
        break
    try:
        عدد = float(ورودی)
        مجموع += عدد
    except ValueError:
        print("لطفاً عدد معتبر وارد کنید!")
print(f"مجموع اعداد: {مجموع}")

# حلقه با شرط ترکیبی
کلمه = ""
while کلمه != "پایان":
    کلمه = input("کلمه وارد کنید (پایان برای خاتمه): ")
    print(f"شما وارد کردید: {کلمه}")
